
# Hardware do projeto

## Componentes utilizados 

- ESP32 (simulação na plataforma Wokwi)
- Sensor LDR (sensor de luminosidade)
- LED (atuador)
- Resistor (limitação de corrente)
- Protoboard (simulaçã)
- Jumpers (conexões)


## Descrição 
 
O ESP32 executa a leitura do sensor LDR, que detecta a intensidade de luz no ambiente. Com base nessa leitura, é LED é adiconado automático.

## Funcionamento

- Baixa luminosidade - LED ligado
- Alta luminosidade - LED desligado

## Plataforma

Projeto desenvolvido e testado utilizando a plataforma de simulação do Wokwi.
